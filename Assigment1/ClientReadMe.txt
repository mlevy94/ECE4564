This is the readme for client.py

Call:
python3 client.py

Dependencies:
tweepy

Usage:
if any tweets are sent to this account with the foramat "@4564Team_13 ##Address:port_"question"" the streamlistener picks up the tweet and passes it to a wolfram alpha server that looks for answers to the question. if the answer is found the account tweets a tweet with the format @username Team_13 "answer".
